
# Calculadora de IMC
Trabalho da faculdade

Uma calculadora de IMC desenvolvida com **HTML**, **CSS** e **JavaScript**, utilizando também o **Bootstrap** para estilização dos alertas.

O projeto calcula o Índice de Massa Corporal (IMC) do usuário e exibe:

* O valor do IMC
* A classificação do peso
* Uma tabela de referência
* Destaque automático da classificação correspondente

---

* HTML5
* CSS3
* JavaScript
* Bootstrap 5

# Funcionalidades

✅ Cálculo automático do IMC
✅ Classificação do resultado
✅ Interface responsiva
✅ Tabela de referência IMC
✅ Destaque visual da classificação
✅ Estilo moderno e organizado

---

# Fórmula utilizada

O cálculo do IMC utiliza a seguinte fórmula:

IMC = \frac{peso}{altura^2}

Exemplo:

* Peso: 70kg
* Altura: 1.75m

Resultado:

IMC = \frac{70}{1.75^2}

# Tabela de classificação

| IMC            | Classificação      |
| -------------- | ------------------ |
| Menor que 18,5 | Baixo peso         |
| 18,5 até 24,9  | Peso normal        |
| 25 até 29,9    | Sobrepeso          |
| 30 até 34,9    | Obesidade Grau I   |
| 35 até 39,9    | Obesidade Grau II  |
| Maior que 40   | Obesidade Grau III |
